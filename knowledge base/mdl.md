# Markdown lint tool

Tool to check markdown files and flag style issues.

## Table of contents <!-- omit in toc -->

1. [TL;DR](#tldr)

## TL;DR

```sh
# Install.
gem install mdl

# Check files.
mdl README.md
mdl path/to/dir/

# Ignore a specific rule.
mdl -r "~MD013" README.md
```
