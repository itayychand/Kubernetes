# Iw

## TL;DR

```shell
# Scan for available wireless networks.
iw dev wlp scan

# Join an open wireless network.
iw dev wlp connect SSID

# Disconnect from the current network.
iw dev wlp disconnect

# Show information about the current connection.
iw dev wlp link
```
