# Pi-hole

## TL;DR

```sh
# One-step automated install.
curl -sSL 'https://install.pi-hole.net' | bash

# Update Graviton's DB.
pihole -g

# Check when Graviton's DB has been updated.
stat /etc/pihole/gravity.db
```
