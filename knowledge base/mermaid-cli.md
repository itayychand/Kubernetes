# mermaid-cli

CLI for mermaid. Takes a mermaid definition file as input and generates an `svg`, `png` or `pdf` file as output.

## TL:DR

```sh
# Installation.
npm install -g '@mermaid-js/mermaid-cli'

# Convert mermaid files to SVG.
mmdc -i 'input.mmd' -o 'output.svg'
```
