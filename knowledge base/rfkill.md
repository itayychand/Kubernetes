# rfkill

1. [TL;DR](#tldr)
2. [Sources](#sources)

## TL;DR

```sh
# List devices.
rfkill list

# Block a device.
rfkill block 'wifi'
rfkill block 'id'

# Unblock a device.
rfkill unblock 'bluetooth'
rfkill unblock 'id'
```
